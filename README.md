# Dynamic To-Do List

**Author:** Rohit Kumar  
**License:** MIT

## Overview
A clean, responsive web-based to-do list that supports adding, editing, deleting, and marking tasks as completed. Designed for easy local setup and quick customization.

## Features
- **Add tasks** with validation to prevent empty entries  
- **Edit tasks** inline and save changes  
- **Delete tasks** dynamically  
- **Mark tasks complete** with a checkbox and visual strikethrough  
- **Counters** for pending and completed tasks  
- Responsive layout that stacks the input vertically for a natural website form

## Files
- **index.html** — main HTML structure and header comments  
- **style.css** — styling and responsive layout  
- **script.js** — JavaScript for task management and UI interactions

## Setup
1. Clone or download the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

## Usage
1. Type a task into the input field.
2. Click **Add Task** or press **Enter** to add it.
3. Use **Edit** to modify a task and **Save** to confirm changes.
4. Click the checkbox to mark a task complete; click **Delete** to remove it.
5. Watch the **Pending** and **Completed** counters update automatically.

## Customization Tips
- Enable persistence by adding `localStorage` logic in `script.js`.
- Change colors and spacing in `style.css` to match your brand.
- Add keyboard shortcuts or drag-and-drop ordering for advanced UX.
